# pomCoder 项目说明

## 项目概述

pomCoder 是一个基于 Ollama + RAG 的智能代码生成系统，提供简洁的Web界面和OpenAI兼容的API接口。

## 核心功能

1. **代码生成** - 基于自然语言生成代码
2. **AI对话** - 支持多轮对话和历史记录
3. **代码切片** - 自动扫描和切片代码文件
4. **知识库管理** - RAG知识库构建和管理
5. **OpenAI API** - 兼容OpenAI的API接口

## 技术栈

- **后端**: FastAPI + Python
- **前端**: 原生HTML/CSS/JavaScript
- **LLM**: Ollama (deepseek-coder:6.7b)
- **向量检索**: FAISS + BM25
- **Embedding**: sentence-transformers/all-MiniLM-L6-v2

## 快速启动

```bash
# 1. 启动Ollama
ollama serve

# 2. 下载模型
ollama pull deepseek-coder:6.7b

# 3. 启动服务
python app_full.py
```

访问：http://localhost:58761

## 配置说明

### 默认配置
- 端口：58761
- API密钥：pom-0721
- 模型：deepseek-coder:6.7b
- Ollama地址：http://localhost:11434

### 修改配置
编辑 `config.yaml` 或使用命令行参数：
```bash
python app_full.py --model <模型名> --port <端口>
```

## API使用

### OpenAI兼容API

```bash
curl http://localhost:58761/v1/chat/completions \
  -H "Authorization: Bearer pom-0721" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "pomcoder",
    "messages": [{"role": "user", "content": "写一个快速排序"}]
  }'
```

### 代码生成API

```bash
curl http://localhost:58761/api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "query": "写一个Python快速排序函数",
    "temperature": 0.2
  }'
```

## 工作流程

### 1. 代码切片 → 知识库构建
```
扫描代码目录 → 生成切片JSON → 构建向量索引 → RAG检索
```

### 2. AI对话
```
用户输入 → (可选)RAG检索 → LLM生成 → 返回结果
```

### 3. 代码生成
```
需求描述 → Query重写 → RAG检索 → Prompt构造 → 代码生成
```

## 特色功能

1. **前后端分离** - 清晰的架构，易于维护
2. **简洁UI** - 白色主题，专业简洁
3. **知识库叠加** - 支持增量更新知识库
4. **对话持久化** - 自动保存对话历史
5. **OpenAI兼容** - 可直接替换OpenAI API

## 性能优化

- 模型预加载到GPU
- 查询缓存机制
- 流式读写大文件
- NPU加速支持（可选）

## 故障排除

### 端口被占用
```bash
python kill_port_58761.py
```

### 模型未加载
检查Ollama服务是否运行，模型是否下载

### 知识库为空
运行代码切片和知识库构建流程
