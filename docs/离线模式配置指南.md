# 离线模式配置指南

## 📋 问题说明

在断网环境下运行时，系统会尝试从Hugging Face下载embedding模型，导致以下错误：

```
MaxRetryError("HTTPSConnectionPool(host='huggingface.co', port=443): 
Max retries exceeded with url: /sentence-transformers/all-MiniLM-L6-v2/resolve/main/modules.json
```

## ✅ 解决方案（V3.6.2强化版）

### 🚀 快速启动（推荐）

使用专用的离线启动脚本：

**Windows**:
```bash
start_offline.bat
```

**Linux/Mac**:
```bash
chmod +x start_offline.sh
./start_offline.sh
```

**Python脚本**:
```bash
python start_offline.py
```

这些脚本会自动设置所有必要的环境变量，确保完全离线运行。

## 📝 详细配置

### 方案1：使用本地缓存（推荐）

系统已自动支持离线模式，会优先使用本地缓存的模型。

#### 步骤1：联网环境下预下载模型

```python
# 在有网络的环境下运行一次，下载模型到本地
from sentence_transformers import SentenceTransformer

model = SentenceTransformer('sentence-transformers/all-MiniLM-L6-v2')
print("模型已下载到本地缓存")
```

#### 步骤2：查看模型缓存位置

**Windows**:
```
C:\Users\你的用户名\.cache\huggingface\hub\
或
C:\Users\你的用户名\.cache\torch\sentence_transformers\
```

**Linux/Mac**:
```
~/.cache/huggingface/hub/
或
~/.cache/torch/sentence_transformers/
```

#### 步骤3：离线使用

断网后，系统会自动使用本地缓存，无需额外配置。

---

### 方案2：手动指定本地模型路径

如果你已经有下载好的模型，可以直接使用本地路径。

#### 修改配置

编辑 `knowledge_base/config.json`：

```json
{
  "embedding_model": "./models/all-MiniLM-L6-v2",
  "num_snippets": 1000,
  "embedding_dim": 384
}
```

#### 模型目录结构

```
./models/all-MiniLM-L6-v2/
├── config.json
├── pytorch_model.bin
├── tokenizer.json
├── tokenizer_config.json
├── vocab.txt
└── modules.json
```

### 方案3：使用镜像源（中国大陆用户）

如果需要在线下载，可以使用国内镜像源。

#### 设置环境变量

**Windows (PowerShell)**:
```powershell
$env:HF_ENDPOINT = "https://hf-mirror.com"
```

**Linux/Mac (Bash)**:
```bash
export HF_ENDPOINT=https://hf-mirror.com
```

#### 永久设置

**Windows**:
1. 右键"此电脑" → 属性 → 高级系统设置
2. 环境变量 → 新建系统变量
3. 变量名：`HF_ENDPOINT`
4. 变量值：`https://hf-mirror.com`

**Linux/Mac**:
```bash
# 添加到 ~/.bashrc 或 ~/.zshrc
echo 'export HF_ENDPOINT=https://hf-mirror.com' >> ~/.bashrc
source ~/.bashrc
```

---

### 方案4：完全离线部署

如果需要在完全离线的环境中部署，需要提前准备。

#### 步骤1：在联网环境下载模型

```bash
# 安装依赖
pip install sentence-transformers

# 下载模型
python -c "from sentence_transformers import SentenceTransformer; SentenceTransformer('sentence-transformers/all-MiniLM-L6-v2')"
```

#### 步骤2：打包模型缓存

**Windows**:
```powershell
# 压缩缓存目录
Compress-Archive -Path "$env:USERPROFILE\.cache\huggingface" -DestinationPath huggingface_cache.zip
```

**Linux/Mac**:
```bash
# 打包缓存目录
tar -czf huggingface_cache.tar.gz ~/.cache/huggingface/
```

#### 步骤3：在离线环境解压

**Windows**:
```powershell
# 解压到用户目录
Expand-Archive -Path huggingface_cache.zip -DestinationPath "$env:USERPROFILE\.cache\"
```

**Linux/Mac**:
```bash
# 解压到用户目录
tar -xzf huggingface_cache.tar.gz -C ~/
```

## 🔧 代码实现

系统已自动实现离线模式支持：

```python
# src/rag_code_generator/ollama_rag_generator.py

# ✅ 离线模式：优先使用本地缓存
try:
    # 设置环境变量，强制使用本地缓存
    import os
    os.environ['TRANSFORMERS_OFFLINE'] = '1'
    os.environ['HF_DATASETS_OFFLINE'] = '1'
    
    embedding_model = SentenceTransformer(model_name, device="cpu")
    logger.info("  - Embedding模型加载完成（使用本地缓存）")
except Exception as e:
    logger.warning(f"离线模式加载失败: {e}")
    logger.info("  - 尝试在线下载模型...")
    
    # 清除离线模式环境变量
    os.environ.pop('TRANSFORMERS_OFFLINE', None)
    os.environ.pop('HF_DATASETS_OFFLINE', None)
    
    # 尝试在线下载
    embedding_model = SentenceTransformer(model_name, device="cpu")
    logger.info("  - Embedding模型加载完成（在线下载）")
```

**工作流程**：
1. 首先尝试离线模式（使用本地缓存）
2. 如果失败，再尝试在线下载
3. 自动降级，无需手动配置

## 📊 模型信息

### 默认模型

- **名称**: `sentence-transformers/all-MiniLM-L6-v2`
- **大小**: ~90MB
- **维度**: 384
- **速度**: 快
- **质量**: 中等

### 替代模型

如果需要更好的效果或更小的体积：

| 模型 | 大小 | 维度 | 速度 | 质量 |
|------|------|------|------|------|
| all-MiniLM-L6-v2 | 90MB | 384 | 快 | 中 |
| all-MiniLM-L12-v2 | 120MB | 384 | 中 | 中+ |
| all-mpnet-base-v2 | 420MB | 768 | 慢 | 高 |
| paraphrase-MiniLM-L3-v2 | 60MB | 384 | 很快 | 低 |

### 更换模型

编辑 `knowledge_base/config.json`：

```json
{
  "embedding_model": "sentence-transformers/all-mpnet-base-v2",
  "num_snippets": 1000,
  "embedding_dim": 768
}
```

## ⚠️ 常见问题

### Q1: 如何确认模型已缓存？

**检查缓存目录**：

**Windows**:
```powershell
dir "$env:USERPROFILE\.cache\huggingface\hub\" | findstr "all-MiniLM-L6-v2"
```

**Linux/Mac**:
```bash
ls ~/.cache/huggingface/hub/ | grep all-MiniLM-L6-v2
```

如果有输出，说明模型已缓存。

---

### Q2: 离线模式仍然尝试联网？

**原因**：
- 模型未完全下载
- 缓存目录权限问题
- 环境变量未生效

**解决**：
```python
# 手动设置离线模式
import os
os.environ['TRANSFORMERS_OFFLINE'] = '1'
os.environ['HF_DATASETS_OFFLINE'] = '1'

# 然后启动服务
python app_full.py
```

---

### Q3: 如何清除模型缓存？

**Windows**:
```powershell
Remove-Item -Recurse -Force "$env:USERPROFILE\.cache\huggingface"
```

**Linux/Mac**:
```bash
rm -rf ~/.cache/huggingface/
```

---

### Q4: 模型下载很慢怎么办？

**使用镜像源**：
```bash
export HF_ENDPOINT=https://hf-mirror.com
pip install sentence-transformers
python -c "from sentence_transformers import SentenceTransformer; SentenceTransformer('sentence-transformers/all-MiniLM-L6-v2')"
```

**或者手动下载**：
1. 访问：https://hf-mirror.com/sentence-transformers/all-MiniLM-L6-v2
2. 下载所有文件到本地目录
3. 使用本地路径加载
