### 缺少的功能

1. **知识库加载**: 没有加载FAISS索引和代码片段
2. **向量检索**: 没有将query转换为向量并检索相似代码
3. **Prompt增强**: 没有将检索到的代码添加到Prompt中
4. **RAG流程**: 完整的RAG流程未实现

## 解决方案

### 方案1: 快速修复（推荐）

修改 `OllamaRAGGenerator` 添加知识库检索功能：

```python
class OllamaRAGGenerator:
    def __init__(
        self,
        model_name: str = "deepseek-coder:6.7b",
        ollama_url: str = "http://localhost:11434",
        knowledge_base_path: Optional[str] = "./knowledge_base",
        ...
    ):
        # 现有初始化...
        
        # 添加知识库检索器
        if knowledge_base_path and Path(knowledge_base_path).exists():
            logger.info("5/5 加载知识库...")
            self.retriever = self._load_knowledge_base(knowledge_base_path)
            logger.info("✓ 知识库加载完成")
        else:
            logger.warning("知识库未找到，将使用纯生成模式")
            self.retriever = None
    
    def _load_knowledge_base(self, kb_path: str):
        """加载知识库"""
        import json
        import faiss
        import numpy as np
        
        kb_path = Path(kb_path)
        
        # 加载FAISS索引
        index = faiss.read_index(str(kb_path / "faiss_index.bin"))
        
        # 加载代码片段
        with open(kb_path / "snippets.json", "r", encoding="utf-8") as f:
            snippets = json.load(f)
        
        # 加载Embedding模型（用于query向量化）
        from sentence_transformers import SentenceTransformer
        embedding_model = SentenceTransformer(
            "sentence-transformers/all-mpnet-base-v2"
        )
        
        return {
            "index": index,
            "snippets": snippets,
            "embedding_model": embedding_model
        }
    
    def generate(self, query: str, ...):
        # 1. Query重写
        query_ctx = self.query_rewriter.rewrite(query)
        
        # 2. 检索知识库（新增）
        retrieved_codes = []
        if self.retriever:
            # 生成query向量
            query_embedding = self.retriever["embedding_model"].encode([query])[0]
            
            # 检索相似代码
            distances, indices = self.retriever["index"].search(
                query_embedding.reshape(1, -1).astype('float32'),
                k=3  # 返回Top 3
            )
            
            # 获取代码片段
            for idx in indices[0]:
                if idx < len(self.retriever["snippets"]):
                    snippet = self.retriever["snippets"][idx]
                    retrieved_codes.append(snippet)
        
        # 3. 构造增强Prompt（修改）
        if retrieved_codes:
            # 添加参考代码到Prompt
            reference_code = "\n\n".join([
                f"参考代码{i+1}:\n```python\n{code['code']}\n```"
                for i, code in enumerate(retrieved_codes)
            ])
            
            user_prompt = f"""请根据以下需求生成代码：

需求描述：
{query_ctx.rewritten_query}

参考以下代码示例：
{reference_code}

请生成完整的、可运行的代码，包含必要的注释。
"""
        else:
            # 原有的Prompt
            user_prompt = f"""请根据以下需求生成代码：

需求描述：
{query_ctx.rewritten_query}

请生成完整的、可运行的代码，包含必要的注释。
"""
        
        # 4. 调用Ollama生成
        generated_code = self.ollama_generator.generate(
            prompt=user_prompt,
            ...
        )
        
        return {
            'code': generated_code,
            'retrieved_codes': retrieved_codes,  # 返回检索到的代码
            ...
        }
```

## 为什么会这样

### 开发历史

1. **初期**：实现了基础的Ollama代码生成
2. **中期**：添加了知识库构建功能
3. **当前**：知识库和生成器是分离的

### 架构问题

```
知识库构建 (build_knowledge_base_npu.py)
    ↓
知识库文件 (knowledge_base/)
    ↓
    ❌ 没有连接
    ↓
代码生成器 (OllamaRAGGenerator)
```

### 需要的连接

```
知识库构建 (build_knowledge_base_npu.py)
    ↓
知识库文件 (knowledge_base/)
    ↓
    ✓ 加载到生成器
    ↓
代码生成器 (OllamaRAGGenerator)
    ↓
检索 → 增强Prompt → 生成
```

import json
import faiss
import numpy as np

index = faiss.read_index("knowledge_base/faiss_index.bin")
with open("knowledge_base/snippets.json", "r", encoding="utf-8") as f:

print(f"索引大小: {index.ntotal}")

queries = [

    "计算斐波那契数列",
    "判断素数"
]

for query in queries:
    print(f"\n查询: {query}")
    query_vec = model.encode([query])[0]
    distances, indices = index.search(
        query_vec.reshape(1, -1).astype('float32'),

        if idx < len(snippets):

            print(f"  {i+1}. {snippet.get('name', 'unknown')}")
            print(f"     距离: {distances[0][i]:.4f}")
