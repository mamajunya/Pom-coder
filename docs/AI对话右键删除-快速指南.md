# AI对话右键删除 - 快速指南

## 🚀 快速开始

### 使用方法

```
1. 打开AI对话助手标签页
2. 在左侧对话列表中找到要删除的对话
3. 右键点击该对话
4. 在弹出菜单中点击"🗑️ 删除对话"
5. 确认删除
6. 完成！对话已删除
```

## ⚠️ 注意事项

```

⚠️ 会删除对话及其所有消息
⚠️ 删除前请确认
```

## 🐛 故障排查

### 问题1：右键菜单不显示

**原因**：
- 浏览器禁用了右键菜单
- JavaScript未加载

**解决**：
```
1. 检查浏览器设置
2. 刷新页面（Ctrl+F5）
3. 查看浏览器控制台错误
```

### 问题2：删除后对话仍然存在

**原因**：
- 网络请求失败
- 服务器错误

**解决**：
```
1. 检查网络连接
2. 刷新页面
3. 查看服务器日志
```

### 问题3：Toast提示不显示

**原因**：
- CSS未加载
- z-index被覆盖

**解决**：
```
1. 刷新页面
2. 清除浏览器缓存
3. 检查CSS文件
```
